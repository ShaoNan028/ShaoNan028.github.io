HTML 基础问题

  1. 什么是 HTML？
  2. HTML 的全称是什么？
  3. HTML 主要用于做什么？
  • HTML 的全称是“HyperText Markup Language”，它是一种用于创建网页的标记语言。
    HTML 主要用于定义和组织网页的结构和内容，通过使用不同的标签和元素来呈现文本、图像、链接等内容，并建立页面的整体结构。
  • HTML 是网页的骨架，如果把一个完整的前端页面比作一个人，那么 HTML 就是这个人的骨架，HTML 决定了人的结构和组成内容，CSS 决定了人的外观和衣着，赋予了页面更多样式上的可塑性，JavaScript 则决定了人的动作和行为，为网页提供了可交互性。

HTML 文档结构

  • HTML 标签就是一种用于描述网页结构和内容的代码，由尖括号包围，通常成对出现，包括开始标签和结束标签。
  • div 标签是基础标签，是一个块级元素，内部可以包含任何类型的内容，可以用于绘制页面的主要结构。
  • span 标签用于包裹文本标签，是行内元素。
  • a 标签是链接标签，用于包裹超链接内容。
  • img 是图片标签，用于将一个图片 URL 在页面中展示出来。
